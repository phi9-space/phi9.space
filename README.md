# phi9.space Website

[![Netlify Status](https://api.netlify.com/api/v1/badges/6ed65619-1dc5-4b7b-a77f-a3681517d7f5/deploy-status)](https://app.netlify.com/projects/phi9/deploys)

A modern, responsive website for phi9.space, built with React, Vite, and Tailwind CSS, focusing on India's defense future through a manifesto and strategic vision.

## Project Structure

Based on the routing defined in the application, the project includes the following main pages:
- **Home**: The landing page of phi9.space.
- **Manifesto**: A detailed page outlining the vision and strategic imperatives for India's defense future.
